MOTIVACIÓN

Esta es una página de prueba para dedmostrar nuestros conocimientos acerca de la programación de páginas webs, esta página esta realizada en la asignatura de funadamentos de la ingeniería informatica, se ha creado el código fuente de la página web utilizando HTML Y CSS.

HTML

El html es un lenguaje de programación creado en 1991 y el signficado de sus letras es: HyperText Markup Language. Es un lenguaje usado para definir la estructura de las páginas web mediante el uso de etiquetas que definen el contenido.

CSS

CSS es uno de los lenguajes más importantes que se utilizan para ordenar las instrucciones referentes a la apariencia de un sitio y presentar los contenidos de una página de forma atractiva. De este modo, HTML se emplea para estructurar el contenido de un sitio, mientras que CSS, para estructurar su presentación.
